# 餐廳清單
使用 Node.js + Express 打造餐廳美食網站

## 功能列表
- 依照餐廳名稱搜尋
- 檢視餐廳詳細資訊包含類別、地址、電話、評分、圖片及 Google Map
- 點選"我的餐廳清單"可返回首頁

### 啟動方式
- 將專案clone到本地端
  ```
  1) git clone http://github.com/Crystal9ong/restaurant-list/git
  ```
  2) 點選 Download ZIP 下載
  ```

- 進到專案資料夾後，安裝工具包
  ```
  cd restaurant_list
  npm install
  ```

- 透過nodemon啟動專案
  ```
  npm run dev
  ```
  或
  ```
  npm start
  ```

- 打開瀏覽器在網址列輸入 localhost:3000 即可瀏覽


### 開發環境
- Node.js: v10.15.0
- Express: v4.17.1
- Express-Handlebars: v5.2.0
